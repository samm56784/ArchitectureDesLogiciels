INSTRUCTIONS À SUIVRE


1. Téléchargez la vidéo fournie sur Moodle et l'enregistrer sous : Ce PC->Disque Local (C:)

2. Aller chercher le projet sur github au lien suivant-> https://github.com/samm56784/ArchitectureDesLogiciels.git

3. Enregistrer le projet sur l'ordinateur

4. Ouvrir Visual Studio, puis ouvrir le projet à l'endroit enregistré sur l'ordinateur

5. Cliquer sur la flèche verte au centre en haut de la fenêtre de Visual Studio où se trouve les écritures -> Débogueur Windows local

6. Un vidéo partira avec une fenêtre noire pour inscrire les commandes

7. Il suffit d'inscrire une commande voulue dans la fenêtre suivie de la touche ENTER

8. Voici les instructions des touches 
P + ENTER – Play ou Pause
A + ENTER – Avance accélérée (1.25x)
R + ENTER – Retour au début
Q + ENTER - Quitter
